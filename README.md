# Visual Time

A single-page timezone visualizer. Four cities displayed as stacked sky-gradient rulers — left edge is now, right edge is 24 hours ahead (6 hours on mobile).

**[Live demo →](https://philask.github.io/visual-time/)**

## Features

- Sky gradient that reflects the actual time of day in each city — deep navy at midnight, golden sunrise, bright noon blue, fiery sunset, purple dusk
- Tick marks every hour (short) and every 3 hours (tall) with time labels
- Midnight divider with date label when the day rolls over within the window
- Click any city name to change it — searches 130+ cities worldwide
- Selections saved in a cookie so your choices persist
- Responsive: 24-hour window on desktop, 6-hour window on portrait mobile

## No dependencies

Single `index.html` file. No build tools, no frameworks, no server required — open directly in a browser.
